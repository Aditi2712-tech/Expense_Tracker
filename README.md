# Expense_Tracker
Tracks expense for college students and provide suggestions on how they can save on unnecessary expenses 

## How to Run

1. Navigate to the `spend-smart` directory:
   ```bash
   cd spend-smart
   ```

2. Install the dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
